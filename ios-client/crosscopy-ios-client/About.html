cross copy was build to fix the ever remaining problems when it comes to quick file transfer between devices. Of course there are many tools helping you to put stuff into the cloud already. But none was satisfying our need for simplicity and spontaneity. Also we think in some cases it's better to not store the data in the cloud.
This software is Open Source and hosted on GitHub. The API is very simple and we would love to see it used for sharing and connecting in other apps.
